# v2ex 野鸡主题 css

## 一、使用方法

打开 v2ex 设置，将 [`v2ex.css`](https://github.com/KyleBing/v2ex-css-kylebing/blob/master/scss/v2ex.css) 内容复制到 `自定义 css` 输入框中，点击 <kbd>保存</kbd> 即可
